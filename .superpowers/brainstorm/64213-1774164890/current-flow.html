Key UX Problems Identified
🚫 "More" goes to Settings
Users expect a menu. They get account settings + danger zone. Confusing and alarming.
🚫 4 screens are invisible on mobile
Alerts, Learn, Orders, Documents unreachable unless user finds dashboard cards or guesses the URL.
🚫 No clear "what next" after onboarding
Dashboard shows a greeting + feature cards but no guided action. New user doesn't know where to start.
🚫 Find page is overloaded
Algolia search + filters + GeM button + live feed + AI summarizer all on one screen. Too much at once.
🚫 Pro gate cliffs
After trial: Bid Generator, Alerts, Documents, Orders all block simultaneously with no soft landing.
🚫 Mobile/desktop nav mismatch
Desktop sees all 9 screens in sidebar. Mobile sees 5. Same app, completely different experience.
